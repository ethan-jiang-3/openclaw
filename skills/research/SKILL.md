# 调研技能

## 工作流

```
1. 确认调研主题和范围
2. 使用 web_search 进行网络搜索
3. 整理搜索结果，提取关键信息
4. 创建飞书文档并写入内容
5. 返回文档链接给用户
```

## 搜索技巧

- 使用 `web_search` 工具，搜索中文内容加 `country: "CN"`
- 复杂主题可以分多个搜索词搜索
- 搜索结果会返回标题、URL和摘要

## 内容组织

整理调研结果时，按以下结构组织：

```markdown
# 调研主题

## 概述

（主题的基本介绍）

## 关键发现

- 要点1
- 要点2
- 要点3

## 详细分析

（各要点的详细说明）

## 参考来源

- [来源1](url1)
- [来源2](url2)
```

## 创建飞书文档

使用 `feishu_doc` 工具：

1. **创建文档**：

```json
{
  "action": "create",
  "title": "调研：{主题}",
  "owner_open_id": "ou_xxx"
}
```

2. **写入内容**（使用markdown格式）：

```json
{
  "action": "write",
  "doc_token": "xxx",
  "content": "# 调研主题\n\n## 概述\n\n..."
}
```

**注意**：markdown表格不被支持，用列表代替。

## 执行步骤

1. 先回复用户确认调研主题："好的，我来调研一下【主题】，预计包含：xxx"
2. 进行搜索（可多次）
3. 整理内容
4. 创建飞书文档并写入
5. 返回文档链接

---

**提示**：搜索功能使用 Tavily API，已配置在系统中。
